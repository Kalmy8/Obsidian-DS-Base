# Backend Basics Course

**Course Description:**
Introduction to backend development covering client-server architecture, HTTP protocol, REST APIs, and backend fundamentals. Designed to provide a solid foundation for understanding how web applications work behind the scenes.

## Course Structure

### Core Lessons

1. **Introduction to Backend**
   - [[Backend Basics (1st lesson)]] - Client-server model, HTTP/HTTPS, requests/responses, status codes, APIs, REST, JSON

2. **API Development**
   - [[Backend Basics (2nd lesson)]] - API design and implementation

3. **Backend Architecture**
   - [[Backend Basics (3rd lesson)]] - Backend architecture patterns

4. **Advanced Topics**
   - [[Backend Basics (4th lesson)]] - Advanced backend concepts

### Planning

- [[plan]] - Course roadmap and learning plan

## Learning Path

The course progresses from basic concepts to advanced backend topics:
1. Understanding client-server architecture
2. Learning HTTP protocol and communication
3. Building REST APIs
4. Exploring backend architecture patterns
5. Advanced backend concepts

## Key Concepts Covered

- **Client-Server Model**: How clients and servers communicate
- **HTTP/HTTPS**: Web communication protocols
- **REST APIs**: Representational State Transfer architecture
- **JSON**: Data interchange format
- **Status Codes**: HTTP response codes
- **Request/Response**: Web communication flow
