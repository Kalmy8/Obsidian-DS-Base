# Python Basics Course

**Course Description:**
Fundamental Python programming course covering core concepts, data types, control structures, functions, and Python-specific features. Designed for beginners to build a solid foundation in Python programming.

1. **Introduction to Python**
   - [[Python Basics (1st lesson)]] - Compiled vs interpreted languages, Python objects, immutable and mutable types

   - [[Python Basics (3rd lesson)]] - Advanced fundamentals
   - [[Python Basics (3.5)]] - Intermediate concepts

3. **Control Structures and Functions**
   - [[Python Basics (4th lesson)]] - Control flow, loops, conditionals

4. **Advanced Topics**
   - [[Python Basics (5th lesson)]] - Advanced Python features
   - [[Python Basics (6th lesson)]] - Further advanced topics

   - [[Python Basics(control work)]] - Control work / assessment

## Learning Path

The course progresses from basic concepts to advanced Python features:
1. Understanding Python as a language (interpreted vs compiled)
2. Learning Python objects and data types
3. Mastering control structures and functions
4. Exploring advanced Python features
5. Practicing with problems and assessments

## Key Concepts Covered

- Python objects and data types
- Immutable vs mutable types
- Control structures (if/else, loops)
- Functions and function definitions
- Python-specific features and idioms

## Related Topics

- Object-oriented programming (OOP)
- Data structures
- Algorithms
- Python standard library
