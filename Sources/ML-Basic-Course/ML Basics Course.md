# ML Basics Course

**Course Description:**
Introduction to Machine Learning covering fundamental concepts, types of ML, and core algorithms. Provides a foundation for understanding how machines learn from data and make predictions.

## Course Structure

### Core Lessons

1. **Introduction to Machine Learning**
   - [[ML Basics (1st) lesson]] - ML definition, types of machine learning (supervised, unsupervised, reinforcement)

2. **ML Fundamentals**
   - [[ML Basics (2nd) lesson]] - Core ML concepts and algorithms

## Learning Path

The course introduces machine learning from the ground up:
1. Understanding what machine learning is
2. Learning different types of ML (supervised, unsupervised, reinforcement)
3. Exploring core ML algorithms and concepts
4. Understanding how ML models learn from data

## Key Concepts Covered

- **Machine Learning Definition**: Learning from data without explicit programming
- **Supervised Learning**: Learning from labeled examples
- **Unsupervised Learning**: Finding patterns in unlabeled data
- **Reinforcement Learning**: Learning through interaction and rewards
- **Model Training**: How ML models learn from data
- **Prediction**: Making predictions on new data
