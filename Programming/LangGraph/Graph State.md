Произвольные "данные", которые [[граф]] модифицирует/хранит в процессе своего исполнения

В базовом варианте хранит историю сообщений:
- Пользователя (**HumanMessage**)
- Инструментов (результат их работы, **ToolMessage**)
- Текста, который генерирует LLM (в т.ч. служебного текста для вызова инструментов, **AIMessage**)
